# create( array $itemData, mixed $rewriteIfExist = false )

By default, insert new (by create) Item.

By default, insert new (by create) Item.

It can't overwrite existing item by default. You can get item "id" for creatad item us result this function.

If $item["id"] !== null, item set with that id. If item with same id already exist - method will throw exception, but if $rewriteIfExist = true item will be rewrited.

If $item["id"] is not set or $item["id"]===null, item will be insert with autoincrement PrimryKey.

By default, update existing Item.

By default, update existing Item.

If item with PrimaryKey == $item["id"] is existing in store, item will updete. Filds wich don't present in $item will not change in item in store.
Method will return updated item

If $item["id"] isn't set - method will throw exception.

If item with PrimaryKey == $item["id"] is absent - method will throw exception,
